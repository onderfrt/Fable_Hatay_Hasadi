Üyelik sistemi henüz aktif değil

Üyelik altyapısı kuruldu ancak Supabase bağlantısı yapılandırılmadı. Aktifleştirmek için:

  1. supabase.com üzerinde ücretsiz bir proje oluşturun.
  2. supabase/schema.sql dosyasını SQL Editor'de çalıştırın.
  3. Project Settings → API sayfasındaki URL ve anon key değerlerini js/supabase-config.js dosyasına yapıştırın.

Detaylı adımlar için repodaki SUPABASE-KURULUM.md dosyasına bakın. Bu süreçte sepet ve WhatsApp siparişleri normal şekilde çalışmaya devam eder.
